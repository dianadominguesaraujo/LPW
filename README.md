Projeto de site utilizando HTML e CSS criado para a disciplina de Linguagem e Programação Web.

O site possui página index com inúmeras propriedades personalizadas via CSS, menu com página ativa, botão de topo, header, sidebar e footer estilizados. Nas demais páginas foram desenvolvidas lista, tabela e formuário.
A página web tem a temática "Vegana da Tattoo", que é um primeiros projetos fotográficos produzidos pela desenvolvedora.
